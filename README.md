# matlab_music
用matlab演奏音乐

利用matlab的sound函数，结合十二平均律等乐理知识，参考乐曲简谱，便能演奏出相应乐曲。

目前实现的曲子：
- 千秋——古剑奇谭3主题曲
- 蜂鸟——吴青峰

参考：
1. https://www.bilibili.com/video/av46176249
2. https://github.com/howardcorleone/Matlab-Music-Play
